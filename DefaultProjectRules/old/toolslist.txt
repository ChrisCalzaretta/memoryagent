## üîß Tool Parameter Reference

### `orchestrate_task` Parameters (coding-orchestrator)
```
task: string (required) - The coding task description
context: string - Project context (auto-detected from workspace)
workspacePath: string - Path to workspace (auto-detected)
background: boolean (default: true) - Run as background job
maxIterations: integer (default: 5) - Max coding/validation loops
minValidationScore: integer (default: 8) - Required score to pass (0-10)
```

**Example:**
```
orchestrate_task(
  task: "Create a CachingService with Redis support",
  maxIterations: 3,
  minValidationScore: 8
)
‚Üí Returns { jobId: "abc123", status: "Running" }
```

### `get_task_status` Parameters (coding-orchestrator)
```
jobId: string (required) - The job ID from orchestrate_task
```

### `index` Parameters
```
scope: "file" | "directory" | "reindex" | "status"
path: string (required)
context: string (required)
recursive: boolean (for directory, default: true)
removeStale: boolean (for reindex, default: true)
background: boolean (run in background to avoid timeout, default: false)
jobId: string (check status of background job with scope='status')
```

**üí° For large projects**: Use `background: true` to avoid timeout:
```
index(scope: "reindex", path: ".", context: "myproject", background: true)
‚Üí Returns job ID immediately, continues indexing in background

index(scope: "status", jobId: "abc12345", path: ".", context: "myproject")
‚Üí Check job progress
```

### `validate` Parameters
```
context: string (required, except for list_best_practices)
scope: "best_practices" | "security" | "pattern_quality" | "anti_patterns" | "project" | "list_best_practices" | "all"
patternId: string (required for pattern_quality)
minSeverity: "low" | "medium" | "high" | "critical"
includeAutoFix: boolean (default: true)
includeLegacy: boolean (for anti_patterns, default: true)
```

### `transform` Parameters
```
type: "page" | "css" | "analyze_css" | "learn_pattern" | "apply_pattern" | "detect_components" | "extract_component" | "list_patterns"
sourcePath: string (required for most types)
outputPath: string (optional)
context: string (for list_patterns filtering)
# Type-specific options: extractComponents, modernizeCSS, addErrorHandling, addLoadingStates, addAccessibility, generateVariables, modernizeLayout, addResponsive
```

### `manage_plan` Parameters
```
action: "create" | "get_status" | "update_task" | "complete" | "search" | "validate_task"
planId: string (required for get_status, update_task, complete, validate_task)
taskId: string (for update_task, validate_task)
status: string (for update_task): Pending, InProgress, Blocked, Completed, Cancelled
# For action='create':
context: string (required)
name: string (required)
tasks: array of {title, description, orderIndex} (required)
includeRecommendations: boolean (default: false)
maxRecommendations: number (default: 10)
```

### `manage_todos` Parameters
```
action: "add" | "search" | "update_status"
# For action='add':
context: string (required)
title: string (required)
description: string (optional)
priority: "Low" | "Medium" | "High" | "Critical" (default: Medium)
filePath: string (optional)
lineNumber: number (optional)
assignedTo: string (optional)
# For action='update_status':
todoId: string (required)
status: "Pending" | "InProgress" | "Completed" | "Cancelled"
# For action='search':
context: string, priority: string, todoStatus: string, assignedTo: string (all optional filters)
```

### `get_insights` Parameters
```
category: "patterns" | "prompts" | "tools" | "sessions" | "domains" | "recalculate" | "all"
context: string (required for sessions, domains, recalculate)
limit: number (default: 10)
# For category='domains':
filePath: string (required)
content: string (optional - file content)
```
